# –ú–∞—Å—Å–∏–≤—ã

üí°**–ú–∞—Å—Å–∏–≤** - —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã—Ö –ø–æ –∏–Ω–¥–µ–∫—Å—É. 
–û–¥–Ω–∞–∫–æ, –≤ Python –º–∞—Å—Å–∏–≤—ã, –æ–±—ã—á–Ω–æ, —Ä–µ–∞–ª–∏–∑—É—é—Ç—Å—è —á–µ—Ä–µ–∑ –¥–≤–∞ –ø–æ–ø—É–ª—è—Ä–Ω—ã—Ö —Ç–∏–ø–∞ —Å—Ç—Ä—É–∫—Ç—É—Ä: `—Å–ø–∏—Å–∫–∏` –∏ –º–∞—Å—Å–∏–≤—ã –∏–∑ –º–æ–¥—É–ª—è `array`.

**üìã–û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ –º–∞—Å—Å–∏–≤–∞–º–∏/—Å–ø–∏—Å–∫–∞–º–∏**
  - –î–æ–±–∞–≤–ª–µ–Ω–∏–µ: `.append()`, `.insert()`.
  - –£–¥–∞–ª–µ–Ω–∏–µ: `.remove()`, `.pop()`.
  - –ü–æ–∏—Å–∫ —ç–ª–µ–º–µ–Ω—Ç–∞: `.index()`.
  - –î–ª–∏–Ω–∞ –º–∞—Å—Å–∏–≤–∞: `len()`.
  - –°—Ä–µ–∑—ã: `my_list[start:end]`- –∏–∑–≤–ª–µ—á–µ–Ω–∏–µ —á–∞—Å—Ç–∏ –º–∞—Å—Å–∏–≤–∞.

## –°–ø–∏—Å–∫–∏
üí°**–°–ø–∏—Å–∫–∏** - –Ω–∞–∏–±–æ–ª–µ–µ —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –∫–æ–ª–ª–µ–∫—Ü–∏–π —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –û–Ω–∏ —è–≤–ª—è—é—Ç—Å—è –≥–∏–±–∫–∏–º–∏, —Ç–∞–∫ –∫–∞–∫ –º–æ–≥—É—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤ (—á–∏—Å–ª–∞, —Å—Ç—Ä–æ–∫–∏, –¥—Ä—É–≥–∏–µ —Å–ø–∏—Å–∫–∏ –∏ —Ç. –¥.). –ù–∞–ø—Ä–∏–º–µ—Ä:
```python
# –°–æ–∑–¥–∞–Ω–∏–µ —Å–ø–∏—Å–∫–∞

# –î–æ—Å—Ç—É–ø –∫ —ç–ª–µ–º–µ–Ω—Ç–∞–º —Å–ø–∏—Å–∫–∞ 
print(my_list[0]) # 1

# –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞

# –£–¥–∞–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞

# –ü—Ä–∏–º–µ—Ä –∏—Ç–µ—Ä–∏—Ä–æ–≤–∞–Ω–∏—è –ø–æ —Å–ø–∏—Å–∫—É 
for elem in my_list: 
	print(elem) # –í—ã–≤–æ–¥: 1 20 3 5 6
```

## –ú–æ–¥—É–ª—å `array`
–ï—Å–ª–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —Ä–∞–±–æ—Ç–∞—Ç—å —Å –º–∞—Å—Å–∏–≤–æ–º, –≤ –∫–æ—Ç–æ—Ä–æ–º –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –æ–¥–Ω–æ–≥–æ —Ç–∏–ø–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ç–æ–ª—å–∫–æ —á–∏—Å–ª–∞), –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–æ–¥—É–ª—å `array`, –∫–æ—Ç–æ—Ä—ã–π —Ä–µ–∞–ª–∏–∑—É–µ—Ç –Ω–∞—Å—Ç–æ—è—â–∏–µ –º–∞—Å—Å–∏–≤—ã, –±–æ–ª–µ–µ –æ–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –¥–ª—è —á–∏—Å–ª–æ–≤—ã—Ö –¥–∞–Ω–Ω—ã—Ö. –ù–∞–ø—Ä–∏–º–µ—Ä: 
```python
import array 

# –°–æ–∑–¥–∞–Ω–∏–µ –º–∞—Å—Å–∏–≤–∞ —Å —Ü–µ–ª—ã–º–∏ —á–∏—Å–ª–∞–º–∏ 
my_array = array.array('i', [1, 2, 3, 4, 5]) 

# –î–æ—Å—Ç—É–ø –∫ —ç–ª–µ–º–µ–Ω—Ç–∞–º

# –ò–∑–º–µ–Ω–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ 
my_array[1] = 20 

# –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ 
my_array.append(6) 

# –£–¥–∞–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞

# –ü—Ä–∏–º–µ—Ä –∏—Ç–µ—Ä–∏—Ä–æ–≤–∞–Ω–∏—è –ø–æ –º–∞—Å—Å–∏–≤—É

	print(elem)
```

**üìã–ö–æ–¥–∏—Ä–æ–≤–∫–∏ —Ç–∏–ø–æ–≤ –¥–∞–Ω–Ω—ã—Ö –¥–ª—è –º–∞—Å—Å–∏–≤–∞**:

## `NumPy` –º–∞—Å—Å–∏–≤—ã
–î–ª—è –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã—Ö –≤—ã—á–∏—Å–ª–µ–Ω–∏–π —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –º–∞—Å—Å–∏–≤—ã –∏–∑ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ `NumPy`, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ —Ä–∞–±–æ—Ç–∞—Ç—å —Å –º–Ω–æ–≥–æ–º–µ—Ä–Ω—ã–º–∏ –º–∞—Å—Å–∏–≤–∞–º–∏ –∏ –º–∞—Ç—Ä–∏—Ü–∞–º–∏, –∏—Å–ø–æ–ª—å–∑—É—è –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏. –ù–∞–ø—Ä–∏–º–µ—Ä:
