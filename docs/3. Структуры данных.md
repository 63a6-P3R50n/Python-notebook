# –°—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö

üí°**–°—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö** - `—Å—Ç–µ–∫`, `–æ—á–µ—Ä–µ–¥—å` –∏ `–∫—É—á–∞` - –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è –æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏–∏ –∏ —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –¥–∞–Ω–Ω—ã–º–∏ –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏. 
–û–Ω–∏ –∏–º–µ—é—Ç —Ä–∞–∑–ª–∏—á–Ω—ã–µ –ø—Ä–∏–Ω—Ü–∏–ø—ã —Ä–∞–±–æ—Ç—ã –∏ –ø—Ä–∏–º–µ–Ω—è—é—Ç—Å—è –¥–ª—è —Ä–µ—à–µ–Ω–∏—è —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –∑–∞–¥–∞—á. –í Python —Å—É—â–µ—Å—Ç–≤—É—é—Ç –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –∏ —Å—Ç–æ—Ä–æ–Ω–Ω–∏–µ —Å–ø–æ—Å–æ–±—ã —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —ç—Ç–∏—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä.

## –°—Ç–µ–∫ (Stack)
üí°**–°—Ç–µ–∫** - —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, —Ä–∞–±–æ—Ç–∞—é—â–∞—è –ø–æ –ø—Ä–∏–Ω—Ü–∏–ø—É `LIFO` (Last In, First Out), —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –ø–æ—Å–ª–µ–¥–Ω–∏–π –¥–æ–±–∞–≤–ª–µ–Ω–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑–≤–ª–µ–∫–∞–µ—Ç—Å—è –ø–µ—Ä–≤—ã–º.

### üìã–û–ø–µ—Ä–∞—Ü–∏–∏ —Å–æ —Å—Ç–µ–∫–æ–º
  - –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞: –æ–ø–µ—Ä–∞—Ü–∏—è –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `push`
  - –ò–∑–≤–ª–µ—á–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞: –æ–ø–µ—Ä–∞—Ü–∏—è –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `pop`.

–í Python —Å—Ç–µ–∫–∏ –ª–µ–≥–∫–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Å–ø–∏—Å–∫–æ–≤ (`list`), –∏—Å–ø–æ–ª—å–∑—É—è –º–µ—Ç–æ–¥—ã `.append()` –∏ `.pop()`. –ù–∞–ø—Ä–∏–º–µ—Ä:
```python
# –†–µ–∞–ª–∏–∑–∞—Ü–∏—è —Å—Ç–µ–∫–∞ —á–µ—Ä–µ–∑ —Å–ø–∏—Å–æ–∫
stack = []

# –î–æ–±–∞–≤–ª—è–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –≤ —Å—Ç–µ–∫ (push)
stack.append(1)
stack.append(2)
stack.append(3)

# –ò–∑–≤–ª–µ–∫–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ —Å—Ç–µ–∫–∞ (pop)
print(stack.pop())  # 3
print(stack.pop())  # 2
```

## –û—á–µ—Ä–µ–¥—å (Queue)
üí°**–û—á–µ—Ä–µ–¥—å** - —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, —Ä–∞–±–æ—Ç–∞—é—â–∞—è –ø–æ –ø—Ä–∏–Ω—Ü–∏–ø—É `FIFO` (First In, First Out), —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –ø–µ—Ä–≤—ã–π –¥–æ–±–∞–≤–ª–µ–Ω–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑–≤–ª–µ–∫–∞–µ—Ç—Å—è –ø–µ—Ä–≤—ã–º.

### üìã–û–ø–µ—Ä–∞—Ü–∏–∏ —Å –æ—á–µ—Ä–µ–¥—å—é
  - –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞: –æ–ø–µ—Ä–∞—Ü–∏—è –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `enqueue`.
  - –ò–∑–≤–ª–µ—á–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞: –æ–ø–µ—Ä–∞—Ü–∏—è –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `dequeue`.

–í Python –æ—á–µ—Ä–µ–¥–∏ –º–æ–∂–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —Å –ø–æ–º–æ—â—å—é –º–æ–¥—É–ª—è `collections.deque`, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –±—ã—Å—Ç—Ä—ã–µ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ –≤—Å—Ç–∞–≤–∫–∏ –∏ —É–¥–∞–ª–µ–Ω–∏—è —Å –æ–±–µ–∏—Ö —Å—Ç–æ—Ä–æ–Ω –æ—á–µ—Ä–µ–¥–∏. –ù–∞–ø—Ä–∏–º–µ—Ä:
```python
from collections import deque

# –°–æ–∑–¥–∞–Ω–∏–µ –æ—á–µ—Ä–µ–¥–∏
queue = deque()

# –î–æ–±–∞–≤–ª—è–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –≤ –æ—á–µ—Ä–µ–¥—å (enqueue)
queue.append(1)
queue.append(2)
queue.append(3)

# –ò–∑–≤–ª–µ–∫–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ –æ—á–µ—Ä–µ–¥–∏ (dequeue)
print(queue.popleft())  # 1
print(queue.popleft())  # 2
```

## –î–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω—è—è –æ—á–µ—Ä–µ–¥—å (Deque)
üí°**Deque (Double-Ended Queue)** - –æ—á–µ—Ä–µ–¥—å, –≤ –∫–æ—Ç–æ—Ä–æ–π –º–æ–∂–Ω–æ –¥–æ–±–∞–≤–ª—è—Ç—å –∏ —É–¥–∞–ª—è—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã –∫–∞–∫ —Å –Ω–∞—á–∞–ª–∞, —Ç–∞–∫ –∏ —Å –∫–æ–Ω—Ü–∞. 

Python –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é `deque` —á–µ—Ä–µ–∑ –º–æ–¥—É–ª—å `collections`. –ù–∞–ø—Ä–∏–º–µ—Ä:
```python
from collections import deque

# –°–æ–∑–¥–∞–Ω–∏–µ –¥–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω–µ–π –æ—á–µ—Ä–µ–¥–∏
d = deque()

# –î–æ–±–∞–≤–ª–µ–Ω–∏–µ –≤ –Ω–∞—á–∞–ª–æ –∏ –∫–æ–Ω–µ—Ü
d.append(1)        # –î–æ–±–∞–≤–ª—è–µ–º –≤ –∫–æ–Ω–µ—Ü
d.appendleft(0)    # –î–æ–±–∞–≤–ª—è–µ–º –≤ –Ω–∞—á–∞–ª–æ

# –£–¥–∞–ª–µ–Ω–∏–µ –∏–∑ –Ω–∞—á–∞–ª–∞ –∏ –∫–æ–Ω—Ü–∞
print(d.pop())     # –£–¥–∞–ª—è–µ–º —Å –∫–æ–Ω—Ü–∞
print(d.popleft()) # –£–¥–∞–ª—è–µ–º —Å –Ω–∞—á–∞–ª–∞
```

## –ö—É—á–∞ (Heap)
üí°**–ö—É—á–∞ (–∏–ª–∏ –ø–∏—Ä–∞–º–∏–¥–∞)** - —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –ø–æ–ª–Ω–æ–µ –±–∏–Ω–∞—Ä–Ω–æ–µ –¥–µ—Ä–µ–≤–æ, –≤ –∫–æ—Ç–æ—Ä–æ–º —Å–æ–±–ª—é–¥–∞–µ—Ç—Å—è –æ—Å–æ–±–æ–µ —Å–≤–æ–π—Å—Ç–≤–æ: *–∫–∞–∂–¥—ã–π —É–∑–µ–ª –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –º–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–µ–Ω (–¥–ª—è min-–∫—É—á–∏) –∏–ª–∏ –±–æ–ª—å—à–µ –∏–ª–∏ —Ä–∞–≤–µ–Ω (–¥–ª—è max-–∫—É—á–∏) –≤—Å–µ—Ö —Å–≤–æ–∏—Ö –ø–æ—Ç–æ–º–∫–æ–≤*. 

### üìã–û–ø–µ—Ä–∞—Ü–∏–∏ —Å –∫—É—á–µ–π:
  - –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞: –æ–ø–µ—Ä–∞—Ü–∏—è –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `push`.
  - –ò–∑–≤–ª–µ—á–µ–Ω–∏–µ –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞: –æ–ø–µ—Ä–∞—Ü–∏—è –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è `pop` (–≤ —Å–ª—É—á–∞–µ min-–∫—É—á–∏).
    
–í Python –∫—É—á–∏ —Ä–µ–∞–ª–∏–∑—É—é—Ç—Å—è —á–µ—Ä–µ–∑ –º–æ–¥—É–ª—å `heapq`. –û–¥–Ω–∞–∫–æ, –º–æ–¥—É–ª—å –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç min-–∫—É—á—É. –ï—Å–ª–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å max-–∫—É—á—É, —Ç–æ –º–æ–∂–Ω–æ –ø—Ä–æ—Å—Ç–æ —Ö—Ä–∞–Ω–∏—Ç—å –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è. –ù–∞–ø—Ä–∏–º–µ—Ä: 
```python
import heapq

# –°–æ–∑–¥–∞–Ω–∏–µ —Å–ø–∏—Å–∫–∞ –¥–ª—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –∫–∞–∫ –∫—É—á–∏
heap = []

# –î–æ–±–∞–≤–ª—è–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –≤ –∫—É—á—É (push)

heapq.heappush(heap, 1)
heapq.heappush(heap, 5)

# –ò–∑–≤–ª–µ–∫–∞–µ–º –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –∫—É—á–∏ (pop)
print(heapq.heappop(heap))  # 1
print(heapq.heappop(heap))  # 5
```
