# –†–µ–≥—É–ª—è—Ä–Ω—ã–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è

üí°**–†–µ–≥—É–ª—è—Ä–Ω—ã–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è (RegEx)** - –º–æ—â–Ω—ã–π –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å —Ç–µ–∫—Å—Ç–æ–º. –û–Ω –ø–æ–∑–≤–æ–ª—è–µ—Ç: –Ω–∞—Ö–æ–¥–∏—Ç—å, –∏–∑–≤–ª–µ–∫–∞—Ç—å, –∑–∞–º–µ–Ω—è—Ç—å –∏ –∏–∑–º–µ–Ω—è—Ç—å —Å—Ç—Ä–æ–∫–∏ –Ω–∞ –æ—Å–Ω–æ–≤–µ —à–∞–±–ª–æ–Ω–æ–≤. –í Python –æ–Ω–∏ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω—ã —á–µ—Ä–µ–∑ –º–æ–¥—É–ª—å `re`.

## –û—Å–Ω–æ–≤–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π
**üìã–°–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã**:
  - `.` - –ª—é–±–æ–π –æ–¥–∏–Ω —Å–∏–º–≤–æ–ª, –∫—Ä–æ–º–µ –Ω–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏.
  - `^` - –Ω–∞—á–∞–ª–æ —Å—Ç—Ä–æ–∫–∏.
  - `$` - –∫–æ–Ω–µ—Ü —Å—Ç—Ä–æ–∫–∏.
  - `\` - —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–∏–º–≤–æ–ª–æ–≤.

**üìã–ö–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã**
  - `*` - `0` –∏–ª–∏ –±–æ–ª–µ–µ –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π.
  - `+` - `1` –∏–ª–∏ –±–æ–ª–µ–µ –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π.
  - `?` - `0` –∏–ª–∏ `1` –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ.
  - `{n}` - —Ä–æ–≤–Ω–æ `n` –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π.
  - `{n,}` - `n` –∏–ª–∏ –±–æ–ª–µ–µ –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π.
  - `{n,m}` - –æ—Ç `n` –¥–æ `m` –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π.

**üìã–ú–Ω–æ–∂–µ—Å—Ç–≤–∞ –∏ –¥–∏–∞–ø–∞–∑–æ–Ω—ã**
  - `[abc]` - –ª—é–±–æ–π –∏–∑ —Å–∏–º–≤–æ–ª–æ–≤ `a`, `b` –∏–ª–∏ `c`.
  - `[a-z]` - –ª—é–±–æ–π —Å–∏–º–≤–æ–ª –æ—Ç `a` –¥–æ `z`.
  - `[^abc]` - –ª—é–±–æ–π —Å–∏–º–≤–æ–ª, –∫—Ä–æ–º–µ `a`, `b` –∏–ª–∏ `c`.

**üìã–ì—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∞ –∏ –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤—ã**
  - `(abc)` - –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∞ —Å–∏–º–≤–æ–ª–æ–≤. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –∫–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–≤ –∫ –≥—Ä—É–ø–ø–µ –∏–ª–∏ –¥–ª—è –∏–∑–≤–ª–µ—á–µ–Ω–∏—è –ø–æ–¥—Å—Ç—Ä–æ–∫.
  - `|` - –ª–æ–≥–∏—á–µ—Å–∫–æ–µ `–ò–õ–ò` (–Ω–∞–ø—Ä–∏–º–µ—Ä, `a|b` ‚Äî —Å–∏–º–≤–æ–ª "a" –∏–ª–∏ "b").

**üìã–ú–µ—Ç–∞—Å–∏–º–≤–æ–ª—ã**
  - `\d` - –ª—é–±–∞—è —Ü–∏—Ñ—Ä–∞ `[0-9]`.
  - `\D` - –ª—é–±–æ–π —Å–∏–º–≤–æ–ª, –∫—Ä–æ–º–µ —Ü–∏—Ñ—Ä—ã.
  - `\w` - –ª—é–±–æ–π —Å–∏–º–≤–æ–ª –∏–∑ –Ω–∞–±–æ—Ä–∞ `[A-Za-z0-9_]`.
  - `\W` - –ª—é–±–æ–π —Å–∏–º–≤–æ–ª, –∫—Ä–æ–º–µ –±—É–∫–≤, —Ü–∏—Ñ—Ä –∏ "_".
  - `\s` ‚Äî –ª—é–±–æ–π –ø—Ä–æ–±–µ–ª—å–Ω—ã–π —Å–∏–º–≤–æ–ª (–ø—Ä–æ–±–µ–ª, —Ç–∞–±—É–ª—è—Ü–∏—è –∏ —Ç. –¥.).
  - `\S` ‚Äî –ª—é–±–æ–π –Ω–µ–ø–µ—á–∞—Ç–∞–µ–º—ã–π —Å–∏–º–≤–æ–ª.

## –û—Å–Ω–æ–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –º–æ–¥—É–ª—è `re`
  - `re.match(pattern, string)` - –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, —Å–æ–≤–ø–∞–¥–∞–µ—Ç –ª–∏ —à–∞–±–ª–æ–Ω —Å –Ω–∞—á–∞–ª–æ–º —Å—Ç—Ä–æ–∫–∏.
  - `re.search(pattern, string)` - –∏—â–µ—Ç –ø–µ—Ä–≤–æ–µ —Å–æ–≤–ø–∞–¥–µ–Ω–∏–µ —à–∞–±–ª–æ–Ω–∞ –≤ —Å—Ç—Ä–æ–∫–µ.
  - `re.findall(pattern, string)` - –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –≤—Å–µ—Ö —Å–æ–≤–ø–∞–¥–µ–Ω–∏–π.
  - `re.sub(pattern, repl, string)` - –∑–∞–º–µ–Ω—è–µ—Ç –≤—Å–µ –≤—Ö–æ–∂–¥–µ–Ω–∏—è —à–∞–±–ª–æ–Ω–∞ –≤ —Å—Ç—Ä–æ–∫–µ –Ω–∞ repl.
  - `re.split(pattern, string)` - —Ä–∞–∑–±–∏–≤–∞–µ—Ç —Å—Ç—Ä–æ–∫—É –ø–æ —à–∞–±–ª–æ–Ω—É.

**üìã–ù–∞–≥–ª—è–¥–Ω—ã–µ –ø—Ä–∏–º–µ—Ä—ã**:
```python
import re

# –ü—Ä–∏–º–µ—Ä 1: –ù–∞–π—Ç–∏ –ø–µ—Ä–≤–æ–µ —Å–æ–≤–ø–∞–¥–µ–Ω–∏–µ

# –ü—Ä–∏–º–µ—Ä 3: –ó–∞–º–µ–Ω–∞

# –ü—Ä–∏–º–µ—Ä 4: –†–∞–∑–±–∏–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏
parts = re.split(r'\s+', '–ü—Ä–∏–≤–µ—Ç, –∫–∞–∫ –¥–µ–ª–∞?')  # –†–∞–∑–±–∏–≤–∞–µ–º —Å—Ç—Ä–æ–∫—É –ø–æ –ø—Ä–æ–±–µ–ª—å–Ω—ã–º —Å–∏–º–≤–æ–ª–∞–º
print(parts)  # –í—ã–≤–µ–¥–µ—Ç: ['–ü—Ä–∏–≤–µ—Ç,', '–∫–∞–∫', '–¥–µ–ª–∞?']
```
